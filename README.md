# Little Noor Public Legal Pages

This folder is prepared for a separate public repository that hosts only legal links for app stores.

Recommended public repo name: `little-noor`  
Recommended host: GitHub Pages

## Content in this folder

- `index.html` (landing page with legal links)
- `privacy-policy/index.html`
- `terms/index.html`
- `assets/style.css`
- `.nojekyll`

## Publish flow (public legal repo only)

1. Create a new local folder outside app code.
2. Copy files from this folder into that new folder.
3. Initialize git and push to public repo `little-noor`.
4. Enable GitHub Pages from `main` branch root.

Expected links:

- `https://<github-username>.github.io/little-noor/privacy-policy/`
- `https://<github-username>.github.io/little-noor/terms/`

## Important

Do not push the application source code to this public repo.
Only these legal files should be public.
